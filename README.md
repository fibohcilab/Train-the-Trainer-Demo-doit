# Train-the-Trainer-Demo-doit
Demo for train the trainer course (non-degree) 2019 with doit shield

# Note

## Pin
### Encoder
  - 2 = CH A
  - 3 = CH B
### Servo & Motor
  - 9 = PWMA
  - 8 = DIRA
  - 6 = PWMB
  - 7 = DIRB
  
### i2c
  - 4 = SDA
  - 5 = SCL

## Arduino Sheild : doit
  - Address = 0x40
