# Train-the-Trainer-Demo-doit
Demo for train the trainer course (non-degree) 2019 with doit shield

# Note

## Pin
### Encoder
  - CH A  = 2
  - CH B  = 3
### Servo & Motor
  - PWMA  = 9
  - DIRA  = 8
  - PWMB  = 6
  - DIRB  = 7
  
### i2c
  - SDA = 4
  - SCL = 5

## Arduino Sheild : doit
  - Address = 0x40
  - https://github.com/SmartArduino/SZDOITWiKi/wiki/Arduino---2-way&16-way-motor&servo-shield
